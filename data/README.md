# Data Directory

This repository does not commit source data snapshots. Place downloaded files in
`data/raw/`, normalized per-source outputs in `data/interim/`, crosswalks and
unmapped-ID logs in `data/crosswalk/`, and assembled edge/node tables in
`data/processed/`.

| Subdirectory | Contents | License Handling |
| --- | --- | --- |
| `raw/` | Upstream snapshots | Preserve upstream names and metadata |
| `interim/` | Normalized source-specific tables | Keep source columns needed for attribution |
| `crosswalk/` | ID maps and unmapped logs | Generated by project code |
| `processed/` | Node/edge tables and HeteroData exports | Derived data; cite all sources |

ChEMBL is CC-BY-SA-3.0, so downstream redistribution must preserve compatible
share-alike attribution. DisGeNET-derived data must not be placed in this tree.
