# Code Of Conduct

This project follows the Contributor Covenant Code of Conduct, version 2.1.

We expect contributors to be respectful, constructive, and careful with claims
about biomedical evidence. Maintainers may remove comments, commits, issues, or
pull requests that are not aligned with these expectations.

Full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
