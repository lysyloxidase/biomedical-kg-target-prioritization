# ADR 0001: Disease Anchor EFO_0004616

## Status

Accepted.

## Decision

Use `EFO_0004616` as the disease anchor for knee osteoarthritis.

## Consequences

This keeps the benchmark phenotype-specific and compatible with Open Targets.
All-OA genetics may be used for biological context, but primary disease-gene
edges should be anchored on knee OA when the source supports it.
